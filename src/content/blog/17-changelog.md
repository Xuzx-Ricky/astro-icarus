# 更新日志

本文档记录了 Astro Icarus 主题的所有版本更新。

## 版本规范

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正

## [2.5.0] - 2025-02-27

### 新增

- 智能日历组件，支持农历、节气、倒计时
- 右键菜单功能
- 鼠标点击动效
- AI 助手组件
- 文章置顶/隐藏功能
- 代码块复制和折叠功能
- 数学公式支持
- 后端管理界面

### 优化

- 一言组件，支持多提供商和自动缓存
- 图片懒加载
- 代码块主题自动切换
- 卡片阴影效果
- 移动端菜单样式

### 修复

- 移动端下拉菜单背景色问题
- 智能日历日期悬停样式
- 文章封面图空白问题

## [2.4.0] - 2025-02-26

### 新增

- 备案信息显示
- 图片自动优化
- HTML 预压缩
- 阅读位置保存

### 优化

- 主题切换同步
- 评论主题切换
- 懒加载图片

### 修复

- 实验性功能配置错误
- 移动端鼠标特效错位

## [2.3.0] - 2025-02-25

### 新增

- 毛玻璃效果
- 搜索功能优化
- 分享功能
- 打赏功能

### 优化

- 导航栏样式
- 页脚样式
- 侧边栏布局

### 修复

- 主题切换问题
- 评论显示问题

## [2.2.0] - 2025-02-22

### 新增

- 评论主题自适应
- Insight 搜索引擎
- 页眉透明效果

### 修复

- pnpm-lock 不匹配
- Giscus 配置更新

## [2.1.0] - 2025-02-20

### 新增

- 初始版本发布
- 8 种评论系统支持
- 多种搜索引擎支持
- 赛博朋克暗色模式
- 三栏响应式布局

## 升级指南

### 从 2.4 升级到 2.5

1. 备份配置文件
2. 更新主题文件
3. 添加新的配置项
4. 测试所有功能

### 从 2.3 升级到 2.4

1. 更新依赖
2. 添加备案配置（可选）
3. 测试构建

### 从 2.2 升级到 2.3

1. 更新配置文件
2. 添加毛玻璃效果配置
3. 测试主题切换
