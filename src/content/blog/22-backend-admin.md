# 后端管理界面使用说明

Astro Icarus 主题提供了图形化的后端管理界面，让你可以更方便地管理博客内容。

## 功能概述

后端管理界面提供以下功能：

1. **文章管理** - 创建、编辑、删除文章
2. **配置编辑** - 可视化编辑主题配置
3. **开发服务器** - 一键启动开发服务器
4. **构建部署** - 构建项目并部署

## 安装

### 依赖安装

```bash
cd admin
pnpm install
```

### 开发模式

```bash
# 启动开发服务器
pnpm dev
```

### 构建桌面版

```bash
# 构建 Windows 版本
pnpm electron:build:win

# 构建 Mac 版本
pnpm electron:build:mac

# 构建 Linux 版本
pnpm electron:build:linux
```

## 使用指南

### 首次使用

1. 打开管理界面
2. 点击"选择项目目录"
3. 选择你的 Astro Icarus 项目目录
4. 进入控制台

### 文章管理

**创建文章**

1. 点击"新建文章"按钮
2. 填写文章信息
3. 在编辑器中撰写内容
4. 点击保存

**编辑文章**

1. 在文章列表中找到要编辑的文章
2. 点击"编辑"按钮
3. 修改内容
4. 点击保存

**删除文章**

1. 在文章列表中找到要删除的文章
2. 点击"删除"按钮
3. 确认删除

### 配置编辑

1. 进入"主题配置"页面
2. 在编辑器中修改配置
3. 点击保存

### 开发服务器

1. 点击"启动服务"按钮
2. 等待服务器启动
3. 在浏览器中访问 `http://localhost:4321`

### 构建部署

1. 点击"开始构建"按钮
2. 等待构建完成
3. 按照部署指南部署到生产环境

## 界面说明

### 控制台

显示博客统计信息：
- 文章总数
- 分类数量
- 标签数量
- 置顶文章数

### 文章列表

显示所有文章的列表：
- 标题
- 分类
- 标签
- 发布时间
- 操作按钮

### 文章编辑器

提供 Markdown 编辑器：
- 编辑/预览切换
- 实时预览
- 语法高亮

### 配置编辑器

提供配置文件编辑器：
- 语法高亮
- 错误提示

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+N | 新建文章 |
| Ctrl+F | 查找 |

## 故障排查

### 无法选择项目

1. 确保选择的是 Astro Icarus 项目目录
2. 确保目录包含 `src/config.ts` 文件

### 文章不显示

1. 检查文章是否在 `src/content/blog/` 目录
2. 检查文件扩展名是否为 `.md`

### 保存失败

1. 检查文件权限
2. 检查磁盘空间
